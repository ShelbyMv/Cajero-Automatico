🏧 Cajero Automático en Java
Un cajero automático multicliente desarrollado en Java con sincronización de hilos para simular retiros bancarios en tiempo real.

🚀 Características
🏦 Simula retiros de una cuenta bancaria compartida.
🔄 Uso de hilos para manejar múltiples clientes al mismo tiempo.
🔐 Implementación de synchronized para evitar condiciones de carrera.
⏳ Simulación de tiempo de transacción con Thread.sleep().
🛠️ Tecnologías utilizadas
Java
Programación concurrente
Manejo de hilos con Runnable y Thread
📌 Mejoras futuras
💰 Depósitos y transferencias.
🖥️ Interfaz gráfica con JavaFX/Swing.
📊 Historial de transacciones almacenado en base de datos.
